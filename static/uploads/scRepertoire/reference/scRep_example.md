# A Seurat Object of 500 Single T cells,

The object is compatible with `contig_list` and the TCR sequencing data
can be added with `combineExpression`. The data is from 4 patients with
acute respiratory distress, with samples taken from both the lung and
peripheral blood. More information on the data can be found in the
following [manuscript](https://pubmed.ncbi.nlm.nih.gov/33622974/).
