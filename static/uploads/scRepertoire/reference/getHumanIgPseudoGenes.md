# Get Human Immunoglobulin pseudogenes

This function returns a character vector of human immunoglobulin
pseudogenes. These are also the genes that are removed from the variable
gene list in
[`quietVDJgenes()`](https://www.borch.dev/uploads/scRepertoire/reference/quietVDJgenes.md).

## Usage

``` r
getHumanIgPseudoGenes()
```

## Value

Character vector of human immunoglobulin pseudogenes.
