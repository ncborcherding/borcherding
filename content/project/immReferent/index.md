immReferent provides a clean R interface to the IMGT (ImMunoGeneTics) and OGRDB (Open Germline Receptor Database) reference databases for immune receptor gene segments and HLA allele sequences. The package standardizes access to germline V, D, and J gene sequences for both TCR and BCR loci, as well as HLA typing references. By providing a programmatic and reproducible way to query these databases, immReferent simplifies workflows in immunogenomics, repertoire analysis, and transplant immunogenetics.
