escape (Easy single-cell analysis platform for enrichment) bridges the gap between single-cell RNA sequencing and pathway-level analysis. The package allows researchers to perform gene set enrichment analysis (GSEA) on individual cells, enabling the characterization of metabolic, signaling, and immune pathways at single-cell resolution. escape integrates directly with Seurat and SingleCellExperiment objects, making it easy to layer pathway scores onto existing single-cell workflows for visualization, clustering, and differential analysis.
