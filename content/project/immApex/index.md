immApex provides a unified interface for applying machine learning and deep learning approaches to adaptive immune receptor sequences. The package includes tools for featurizing T-cell receptor (TCR) and B-cell receptor (BCR) sequences using a variety of encoding schemes, training predictive models for antigen specificity, and benchmarking model performance. immApex is designed to lower the barrier to entry for researchers interested in applying computational approaches to immune receptor repertoire data.
