bHIVE (B-cell Hybrid Immune Virtual Evolution) is an artificial immune system simulator that models the processes of B-cell receptor diversification, affinity maturation, and clonal selection. The package enables researchers to generate synthetic BCR repertoires under controlled evolutionary pressures, providing a framework for benchmarking repertoire analysis tools, testing hypotheses about immune dynamics, and generating training data for machine learning models of B-cell biology.
